	Control:

	text: number only (validation: 0 < m < 10000)
2. Nhân sự:
	a. Quản lý chấm công: (sort/search/filter)
		
//////////		Claude here
		
	b. Thêm/xóa nhân sự (sort/search/filter)
		New, import, export
		Bảng item (1 entry = 1 người, công cụ: select, edit, delete, cancel)
			Edit: tên người, id, chức vụ
		Page: (Pre | 1, 2, 3, ..., 100 | Next)
3. Bãi đậu xe:
	Quản lý nhiều campus: (sort/search/filter)
		Bảng item (1 entry = 1 bãi đậu xe, công cụ: select, edit, delete, cancel)
			Edit: tên bãi đậu, id, max cap, open/close time
		Page: ... như trên
